Administrar todos os dados do sistema:

É aqui onde todos os dados do sistema são administrado pelos funcionários.

Cada cargo possuí seu nível de acesso e ele determina o que o funcionário que está naquele cargo pode fazer em relação a manipulação de dados, variando de 0 (baixo) a 10 (alto).

Na barra ao lado é possível ver todas as opções de manipulação de dados, onde ao escolher listar os dados de alguem tipo de objeto, é possível fazer uma pesquisa por um dado em específico, editar dados ou excluí-los.

Também é possível acessar a documentação API, onde são feitos testes sobre a comunicação com a API e acessar o bando de dados diretamente, porém é necessário uma senha.
